**Real-Time Balance**: Automatically calculates total balance, income, and expenses as you add transactions.
**Data Persistence**: Uses `localStorage` to ensure your data stays saved even after refreshing the page.
**Dynamic History**: A detailed list of all transactions with the ability to delete individual entries.
**Visual Insights**: A CSS-based pie chart (using `conic-gradient`) that updates dynamically to show spending categories.
**Form Validation**: Prevents empty entries or invalid numeric inputs to keep data clean.
